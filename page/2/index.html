女王大人本纪

2015-08-12 于 民和官厅土族中学 | 分类于 彩虹

###女王扫六合
“陈皇后，快来帮我拧水龙头。”
开始听到这样的称呼时，我是拒绝的，但当听说其他五名男生都纷纷被冠上了诸如“李娘娘”，“姚妃”，“岳飞”这样的称号时，我也只能笑笑了。
等等，什么？岳飞？我也要做岳飞，我从小看《说岳全传》长大的！
女王大人听了一直大笑，冲着边上的人就笑话着：“他要做月妃，你都已经是皇后了为什么要还要叫妃……”
明白了岳飞的真正含义后，我只能迎风洒泪，赋诗一首：
女王扫六合，虎视何雄哉。
张口决名分，诸侯尽西来。

晒晒阳光

2015-08-06 于 民和官厅土族中学 | 分类于 彩虹

中午接到去年一个学生电话，说是在办公室门口大树下等我，我知道他家里事比较多，急急忙忙收拾了下，就出去找他。
我朝着大树方向走，看见几个男孩中起身站起一个孩子，个子不高，走再近了一些，认出是宇宇。他简单的穿着件衬衫，模样没多大变化，就是刘海很长。
“怎么头发留这么长啊？”，我问。
“这是我们土族的习俗，家里没了人的话，100天内不能剪身上的东西。”他回答。

他去年上课很认真的，永远抬着头认真听着。我本以为他今年一定会来的，但刚到官亭第一天斗海峰就跟我说了他家里出了事。我去年还知道，他爸爸有些酗酒，醉酒了以后就喜欢打他，他妈妈一直有病，没法干重活。

送别记——总结篇（7月8日）

2015-07-09 于 西安书院 | 分类于 北京

上了车，简单安置了行李，到自己床铺躺下，看到大多是不同个年龄段的人，听着车厢里说着我听不懂的方言，只能拿着手机不断刷大家的消息，群里聊的依旧是小补的combo和梦雨的种子。

想起中午和q哥说的，不知道最后送自己时会是何种心情，只能说，得益于最后送的是梦雨，心情很平静。因为同是西站且发车时间很近，相约着5点半从南门出发。差不多四点多些整好了行李，和大龟道了声别，大龟还在为行李焦头烂额，忙着打各个电话。去隔壁见军团，明哥，曾哥都在整行李，简单的说了几个常见的梗，送了几句话，就出门了。
在食堂二楼要了份饭，扒拉了几口又没吃完，下楼看见军团在接电话，对着宿舍楼拍了张照，5点出头就到南门了。

送别记——7月7日

2015-07-07 于 北交大16号楼 | 分类于 北京

刚拿到外卖没多久，欣欣告诉我她要出门了，我回复让她等我，扒拉了几口饭，拿上《红楼梦魇》赶紧出门。

老远就看到她一个人瘦瘦小小的站在南门口，旁边立着个巨大的行李箱。我接过行李箱，把书递给她，告诉她书里还有一枚史湘云的磁书签。
路上没挤没堵，很快就到了西站，她说已经在南门一个人自拍过了，可还是拉着我在西站入站口又拍了一遍。笑一下，她说。
眼看就要入站，我说还早的话就继续聊会吧，她去买了两瓶水，我放好行李在入站口边上的阴凉处等她。
就算只能在最后的时候聊一聊了，也想要在最后的时候再聊一聊。
我们相互回顾了两人四年间的故事，相互历数了和班里各个同学的交情。
感慨即将离开生活了四年的城市还可能再不回来。
遗憾四年聚会太少。
感慨跟好多同学都还没什么了解。
遗憾最后一次聚会都没和人敞开心扉聊聊天。

最后，我故意走慢了些跟在后面拍了她入站的背影。返程乘地铁时，我突然感觉走得人像是我一样。

送别记——7月6日

2015-07-06 于 北交大16号楼 | 分类于 北京

一早无事。
也就聪聪拿了个粉红的QQ公仔要送我，还跟我比了比四年来谁积攒了1毛多。
也就是阿福拿了两个小和尚让我挑了一个，说是前不久少林寺买的。

很早就知道汉林下午3点要走，虽然自己手上还有好多事情要赶，但到了2点半无论如何是坐不安定了，起身去了113。汉林还在休息，闲着无聊，就和聪聪下了把象棋。期间阿福跑来说耳朵被自己扣破了，就赶紧去了校医院。
到了快50分，把汉林叫醒，让他检查检查行李，汉林带着永远相同的表情擦了擦眼，戴上眼镜查行李。我和聪聪继续下棋，不时撇两眼，看看收拾的怎么样了。
收拾完了就停下棋子立马起身，我和聪聪各抓着一条行李袋的带子拎着走，一路上又免不了吐槽行李重，带子太细不好抓手，然后嘱咐汉林路上要抱着行李走，别破了袋子，到了杭州怎么走会更好，大抵都是没话找话，免得沉默。
车到了，帮着把行李拎上车阶，相互挥了挥手，看着车远去。

送别记——7月5日

2015-07-05 于 北交大16号楼 | 分类于 北京

早上6点多些，和骐哥、聪聪、明哥吃过早饭，回寝室睡觉路上遇到了磊哥，拖着个行李箱，提着几件小物件，说是最后一点东西，搬到三元桥的家里就不用再回来了。相互为能在路上相遇庆幸了一番，磊哥照例的说了说都还在北京，没事出来聚聚，照例的说完呵呵笑了两声。

躺床上已经是7点了，大概是因为去了被褥新铺上凉席的缘故，我翻来覆去也没睡着。9点多的时候辉哥回来了，我反正睡不着，索性也就起来了。问了辉哥知道了他11点走，就一边玩着pad一边等他整完行李。
因为昨晚闹腾这个点大家也基本还在睡着，好容易找来了晓龙帮忙，我们俩就各帮辉哥拎了一大袋行李，一路上抱怨行李怎么出奇的重。
到了东门，车子已经在等了，辉哥简单的介绍了自己8月就回北京，相互拍了拍肩膀，也就走了。
